This folder contains the following files:
    - main.py:  The main function which can be called from the command line
                    $python3 main.py 0 1 2 3 0 1 2 1 etc.
                It will print the results of a cycling board with these
                traps
    - MDP.py:   Contains a short recursive and flexible implementation
                of an mdp

    - dies.py, traps.py board.py:
                These are pretty self-explanatory

    - autoplay.py:
                Contains the agents as well as the 'battle' functions

    - gui.py:   Contains a function to generate figures like the ones
                seen in the report.
                Also contains the code for the figures and values from
                the report
